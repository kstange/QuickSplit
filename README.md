# About QuickSplit

This addon allows you to click an item stack while pressing ALT and automatically split the stack into individual items up to the available space in the container.

## Features

See above!

## Classic Support

I've done limited testing with Classic versions because I don't play them, but I believe everything should work.  If you report bugs or submit patches, I'll do my best to address them.

## Compatibility

This will probably work with any regular container in the game.

## Localization

There is no localization for this addon.
